# Short-Term Crypto Trend Prediction using Deep Learning

## Project Overview
[cite_start]This project aims to predict short-term market trends (upward, downward, or neutral) for major cryptocurrencies like Bitcoin using a hybrid Deep Learning architecture[cite: 6].

## Methodology
[cite_start]The project follows a structured experimental approach[cite: 60]:
1. [cite_start]**Baseline**: Univariate LSTM model predicting closing prices[cite: 22, 48].
2. [cite_start]**Extensions**: Hybrid LSTM-CNN model and sentiment fusion using BERT/VADER[cite: 26, 28].

## Repository Structure
- [cite_start]`/notebooks`: Contains the Google Colab link and .ipynb files.
- [cite_start]`/report`: Technical report (max 5000 words)[cite: 64].
- [cite_start]`/presentation`: PPT for the 15-minute workshop[cite: 65].

## How to Reproduce
1. Open the notebook in Google Colab.
2. Ensure you are using a T4 GPU runtime.
3. [cite_start]Run all cells to download data from Yahoo Finance and train the model[cite: 13, 22].
